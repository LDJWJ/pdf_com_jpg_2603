# PDF Tools

PDF 파일을 이미지로 변환하고 압축할 수 있는 웹 기반 도구입니다.

웹 URL : https://ldjwj.github.io/pdf_com_jpg_2603/

## 기능

### 1. PDF to Image Converter
- PDF 파일을 JPG/PNG 이미지로 변환
- 이미지 품질 조절 (0.1 ~ 1.0)
- 해상도 배율 설정 (1x ~ 4x)
- 페이지 범위 지정 (전체 또는 특정 페이지)
- ZIP 파일로 일괄 다운로드
- 개별 페이지 다운로드

### 2. PDF 압축
- PDF 파일 크기 축소
- 해상도 배율 설정 (0.5x ~ 3x)
- 5단계 이미지 품질 (최소/낮음/중간/높음/최대)
- 원본 및 압축 후 파일 크기 비교
- 절감률 표시

## 사용 방법

1. 브라우저에서 `index.html` 파일 열기
2. 왼쪽 사이드바에서 원하는 도구 선택
3. PDF 파일을 드래그 앤 드롭하거나 파일 선택 버튼 클릭
4. 설정 조절 후 변환/압축 버튼 클릭
5. 완료 후 다운로드

## 사용 기술

- **PDF.js** - PDF 파일 렌더링
- **jsPDF** - PDF 파일 생성 (압축용)
- **JSZip** - ZIP 파일 생성
- **FileSaver.js** - 파일 다운로드

**주요 업데이트: 사이드바 메뉴 + PDF 압축 기능 추가**
